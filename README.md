# Project_OOP

# Изучение C++ (OOP Examples)

Этот репозиторий содержит мои учебные примеры по объектно-ориентированному программированию на C++.

📚 Что реализовано

- Классы и наследование**: базовый класс Character, наследники Scorpion, SubZero, Noob, Cyrax
- Полиморфизм: виртуальная функция specialAttack(), вызов через массив указателей на базовый класс
- Абстрактные классы: Character сделан абстрактным (specialAttack() = 0)
- Система оружия: абстрактный класс Weapon, конкретные виды Sword, Bow, Staff с уникальными способностями
- правление памятью: виртуальные деструкторы, создание и удаление объектов через new/delete

🛠️ Как собрать и запустить

```bash
cd src
g++ learn_OOP.cpp -o learn
./learn
